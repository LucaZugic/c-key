## Notes on Using This Template

1. **One decision per ADR.** If you're making multiple related decisions, either combine them into one or write separate ADRs.

2. **Keep it brief.** Half a page is usually enough. ADRs are reference documents, not novels.

3. **Write in the present tense.** "We use X" not "We will use X" or "We decided to use X."

4. **Include the date.** Decisions have context in time.

5. **Update status when superseded.** Don't delete old ADRs; mark them as superseded and link to the new one.

6. **Number sequentially.** 0001, 0002, etc. Gaps are fine if decisions are removed.
